
### 函数的length 属性 将返回没有指定默认值的参数个数
```
function(a){} // 1
function(a,b,c=5){} // 2
function(a,b) // 2
```

### 箭头函数
- 函数体内部的this对象就是定义时所在的对象，而不是使用时所在的对象
- 不可以当作构造函数 也就是说 不可以使用new命令 否则会抛出一个错误
- 不可以使用arguments对象，该对象在函数体内不存在， 如果要用可以用rest参数代替
- 不可以使用yield命令 因此箭头函数不能用作generator函数
- this的指向是可变的 但在箭头函数中是固定的
